# LLM

Repositorio base para el desarrollo de un LMS orientado a formacion tecnica con e-learning y simuladores virtuales.

## Objetivo inicial

Construir un LMS que tome como documento rector el manual de contenido y requisitos entregado para el Instituto Profesional y Tecnico de Veraguas.

## Base documental

- Manual fuente: `SOFTWARE TERMINOS Y REFERENCIAS e-LEARNING(JUAN).pdf`
- Transcripcion de trabajo: `texto_extraido.md`
- Requisitos base interpretados: `docs/requisitos-base.md`

## Linea de trabajo

- Definir arquitectura funcional del LMS
- Modelar estructura academica: areas, cursos, modulos, practicas y evaluaciones
- Definir estrategia de simuladores:
  - embebidos
  - propios
- Construir backlog por fases
- Implementar la plataforma

## Estado

El repositorio se crea como punto de partida local y remoto para documentacion, prompts, investigacion e implementacion.
